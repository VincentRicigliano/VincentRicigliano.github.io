A from-scratch hash table implementation used to build a word concordance: an index of every word in a text file and the line numbers it appears on. Built at Cal Poly as part of CSC 202, achieving 100% test coverage across 221 statements.

  • Implemented a hash table from scratch using Horner's hash function and quadratic probing for collision resolution
  • Dynamic resizing: table doubles when load factor exceeds 0.5, maintaining O(1) average lookup
  • Concordance builder parses text files, strips punctuation, filters stop words, and indexes every word with its line numbers
